markov_helpers.utilities.rev_seq¶

rev_seq(seq)¶

reverse a sequence, leaving it a sequence if possible

Parameters
seq_ty.Reversible

Either a sequence or a reversible iterator

Returns
rseq_ty.Reversible

If seq is a sequence, this is the sequence in reversed order, otherwise it is a reversed iterator over seq.

Return type

Reversible
