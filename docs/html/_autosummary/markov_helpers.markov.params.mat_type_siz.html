markov_helpers.markov.params.mat_type_siz¶

mat_type_siz(params, states, **kwds)¶

Is process (uniform) ring/serial/… inferred from array shapes

If uniform, we cannot distinguish general, seial and ring without looking at matrix elements.

Parameters
paramsint or ndarray (np,)

Number of rate parameters, or vector of rates.

statesint or ndarray (n,…)

Number of states, or array over states.

Returns
serialbool

Is the rate vector meant for serial_params_to_mat or gen_params_to_mat?

ringbool

Is the rate vector meant for ring_params_to_mat or gen_params_to_mat?

drn: bool

If nonzero, only include transitions in direction i -> i + sgn(drn). Can only determine |drn|, not its sign.

uniformbool

Is the rate vector for *_params_to_mat or uni_*_params_to_mat? * = general, serial or ring.

Return type

Tuple[bool, …]
