calc_peq(rates, luf=False)¶

Calculate steady state distribution.

Parameters
ratesnp.ndarray (…,n,n) or tuple(np.ndarray) ((…,n,n), (…,n,))

Continuous time stochastic matrix or LU factors of inverse fundamental.

Return LU factorisation of inverse transpose fundamental matrix as well? Will not broadcast if True. Default: False.

Returns
peqnp.ndarray (…,n,)

Steady-state distribution.

(z_lu, ipv)tuple(np.ndarray) ((…,n,n),(…,n,))

LU factors of inverse transposed fundamental matrix.

Tuple[ndarray, Tuple[ndarray, …]]
