markov_helpers.markov.params.ring_params_to_mat¶

ring_params_to_mat(params, drn=0, axis=- 1, daxis=0, **kwds)¶

Ring transition matrix from independent parameters.

Parameters
paramsndarray (2n,)

Vector of independent elements, in order: mat_01, mat_12, …, mat_n-2,n-1, mat_n-1,0, mat_0,n-1, mat_10, mat_21, …, mat_n-1,n-2.

drn: int, optional, default: 0

If nonzero, only include transitions in direction i -> i + sgn(drn).

axisint, optional

Axis along which each set of parameters lie, by default -1.

daxisint, optional

Axis to broadcast non-scalar drn over, by default: 0

Returns
matarray (n,n)

Continuous time stochastic matrix. The extra axis in (from,to) is inserted after axis.

See also

ring_subs, ring_mat_to_params
Return type

~Array
