stochastify_d¶

stochastify_d(mat)¶

Make a matrix the generator of a discrete time Markov process by scaling.

Scales each row to make row sums one. Modifies in place, does not return.

Parameters
matla.lnarray (…,n,n)

Square matrix with non-negative elements. Modified in place.
