markov_helpers.graphs.plots.GraphPlots¶

class GraphPlots(graph, pos=None, axs=None, opts=None, **kws)¶

Bases: object

Class for plotting markov process as a graph.

Parameters
graphDiGraph

Graph object describing model. Nodes have attributes key and value. Edges have attributes key, value and pind (if the model was a SynapseParamModel).

posDict[Node, ArrayLike]|None, optional

Place to plot each node, by default None -> opts.layout(graph).

axsmpl.axes.Axes|None, optional

The axes to draw the graph on, by default None -> plt.gca().

optsGraphOptions|None, optional

Options for plotting the graph, by default None -> GraphOptions().

Other keywords passed to `opt` or `nx.draw_networkx_nodes` and
`nx.draw_networkx_edges`.
